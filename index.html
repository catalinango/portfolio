Ajuste - Regression Calculator

This is simple web application I built with Angular.
It purpose is to calculate Linear and Non linear Regressions with the Coefficient of Determination of each regression using the Least Squares Method.
The results are shown in both, mathematical expresions and graphics.

I've incorporated third-party libraries such as, AdminLTE (Bootstrap dashboard), MathJax (math expresions) and ChartJS (graphics).
